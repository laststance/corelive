#### Usage

<Source language="tsx" code={`import { Toggle } from '@/components/ui/toggle'
import { Bold, Italic } from 'lucide-react'

export default function Example() {
  return (
    <div className="flex items-center space-x-2">
      <Toggle>
        <Bold className="size-4" />
        <span>Bold</span>
      </Toggle>
      <Toggle variant="outline">
        <Italic className="size-4" />
        <span>Italic</span>
      </Toggle>
    </div>
  )
}
`} />

#### Props

- `variant` - Controls the visual style: `"default"` or `"outline"`
- `size` - Controls the size: `"sm"`, `"default"`, or `"lg"`
- `pressed` - Whether the toggle is pressed or not
- `defaultPressed` - The default pressed state when initially rendered
- `onPressedChange` - Event handler called when the pressed state changes
- `disabled` - When true, prevents the user from interacting with the toggle
- `asChild` - Change the component to the HTML tag or custom component of the only child
