#### Usage

<Source language="tsx" code={`import { Avatar, AvatarImage, AvatarFallback } from '@/components/ui/avatar'

export default function Example() {
  return (
    <Avatar className="size-12">
      <AvatarImage src="/user.jpg" alt="User" />
      <AvatarFallback>JD</AvatarFallback>
    </Avatar>
  )
}`} />

#### Props

<Description>
  Composes Radix Avatar primitives. Common props: - `className`: string — size
  and shape control - `src` (on `AvatarImage`): string — image source -
  `children` (on `AvatarFallback`): fallback initials or icon
</Description>
