## Tailwind tokens enabled by shadcn/ui globals.css

The following utility classes are powered by CSS variables defined in `src/globals.css`. Use them directly in JSX or MDX examples.

### Core surface and text

- `bg-background` / `text-foreground`
- `bg-card` / `text-card-foreground`
- `bg-popover` / `text-popover-foreground`

### Brand and emphasis

- `bg-primary` / `text-primary-foreground`
- `bg-secondary` / `text-secondary-foreground`
- `bg-muted` / `text-muted-foreground`
- `bg-accent` / `text-accent-foreground`
- `bg-destructive` (destructive foreground is handled per-component)

### Borders, inputs, focus

- `border-border`
- `border-input`
- `ring-ring`

### Charts (theme extension)

- `bg-[var(--chart-1)]`
- `bg-[var(--chart-2)]`
- `bg-[var(--chart-3)]`
- `bg-[var(--chart-4)]`
- `bg-[var(--chart-5)]`

### Sidebar (theme extension)

- `bg-sidebar` / `text-sidebar-foreground`
- `bg-sidebar-primary` / `text-sidebar-primary-foreground`
- `bg-sidebar-accent` / `text-sidebar-accent-foreground`

### Radii tokens

- `rounded-[var(--radius-sm)]`
- `rounded-[var(--radius-md)]`
- `rounded-[var(--radius-lg)]`
- `rounded-[var(--radius-xl)]`

These classes map to OKLCH/HSL variables defined in `:root` and `.dark` within `src/globals.css`.
