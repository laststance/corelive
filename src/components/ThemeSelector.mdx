# ThemeSelector

Two-axis theme picker for CoreLive — pick a **color family** (Warm Cathedral,
Harbor, Grove, Rose Tea, Iris, Graphite) and a **mode** (Light / Dark, plus
System for the default family). Each family card previews the real theme tokens
(surface · accent · text · heatmap ramp) — no single hex dot.

<Primary />

<Controls />

## Components

| Component               | Use Case                                          |
| ----------------------- | ------------------------------------------------- |
| `ThemeSelector`         | Two-axis picker for the Settings page             |
| `ThemeSelectorMenuItem` | Sidebar quick-switch submenu (same axes, compact) |

Both consume the shared `useThemeAxis` hook, which maps a `(family, mode)` pair to
a stored theme id and back.

## Adding a new theme family

1. Add a \`DerivedTheme\` seed (one per light/dark) to \`THEME_REGISTRY\` in
   \`src/lib/themes/registry.ts\`, and its label to \`THEME_FAMILY_LABEL\`. \`THEMES\`,
   \`THEME_META\`, the picker, and \`useThemeAxis\` all derive from the registry —
   never edit them directly.
2. Regenerate the static CSS — the build-time generator emits the colored family's
   token block; the default Warm Cathedral stays hand-authored in \`globals.css\`:

<Source
  language="bash"
  code={`pnpm theme:generate # writes src/lib/themes/generated.css`}
/>
